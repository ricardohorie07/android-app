# PDV Android App

App Android desenvolvido em Kotlin para simular um sistema de caixa (PDV).

## 📱 Funcionalidades
- Adicionar produto
- Calcular total
- Limpar venda

## 🛠️ Tecnologias
- Kotlin
- Jetpack Compose
- Android Studio
